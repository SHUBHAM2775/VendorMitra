## 💡 Core Idea

> Build a simple B2B marketplace to empower **local street food vendors** and **suppliers** by solving raw material sourcing through a trust-based platform.

- Vendors can browse & order **verified raw material bundles** from nearby suppliers.
- Suppliers are listed with **trust metrics** like reviews, ratings, and verification.
- Orders support **bulk buying + local pickup/shared delivery**.

---

## 🧩 Key Features

### 🧑‍🍳 Vendor Dashboard
- 🔐 Login/Signup
- 🧺 Browse raw material bundles (e.g., “Chaat Stall Bundle”)
- 📦 Place quantity-based orders
- 🌟 View supplier ratings and delivery modes

---

## 🎯 Why It’s Valid

- **Real Need:** Local vendors struggle to source and negotiate for raw materials daily.
- **Trust-Driven:** Suppliers are verified + rated, building buyer confidence.
- **Minimal Logistics:** Vendors pick up orders or coordinate low-cost shared delivery.
- **Scalable Model:** Start in one locality, grow city-wise.

## 🚀 Quick Start (for Developers)

1. **Clone the repo**
   git clone https://github.com/your-username/vendor-mitra.git
   cd vendor-mitra


2. **Start Backend**

   cd backend
   npm install
   npm run dev
 

3. **Start Frontend**


   cd frontend
   npm install
   npm start
